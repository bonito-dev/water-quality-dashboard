# Water Quality Monitoring Dashboard

A data pipeline and dashboard analysing water quality trends in Kenya.

## Data Sources
- WHO/UNICEF JMP — household WASH estimates 2000–2030
- World Bank Open Data — sanitation and water stress indicators
